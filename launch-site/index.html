Install in Minutes

Cross-platform support for macOS, Linux, and Windows

1

Clone and Install

terminal
# Clone the repository
git clone https://github.com/yourusername/imgtagplus.git
cd imgtagplus
# Install dependencies
pip install -r requirements.txt
2

Launch the Application

terminal
# Interactive menu (TUI)
imgtagplus
# Or start Web UI directly
imgtagplus --start-server
3

Start Tagging

Open http://127.0.0.1:5000 in your browser, select a folder, and click "Start Tagging"
